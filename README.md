# 🎓 Student Feedback System
A simple full stack web application built using Flask that allows users to submit feedback and view all submitted responses.
---

## 🚀 Features
- Submit feedback with name and message
- Store data using SQLite database
- View all feedback entries
- Clean and user-friendly UI
---

## 🧱 Tech Stack
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Database: SQLite
---

## 📂 Project Structure
flask_feedback/
|
├── app.py
├── database.db
├── templates/
│ ├── form.html
│ └── view.html
---

## ▶️ How to Run
1. Clone the repository
2. Open project folder
3. Install Flask: pip install flask
4. Run the app: python app.py
5. Open in browser: [http://127.0.0.1:5000](http://127.0.0.1:5000/)

## 💡 How It Works
- User submits feedback through a form
- Data is stored in SQLite database
- Feedback is displayed on a separate page
---

## 🎯 Purpose
This project demonstrates basic full stack development using Flask including routing, database handling, and frontend integration.
